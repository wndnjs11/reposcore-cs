# 이슈 작업 선점 관련 규정 가이드

## 1. 작업 선점 규칙
- 새로운 이슈가 등록된 후 **24시간 이내**에  
  이슈 작성자를 제외한 다른 사람이 아래와 같은 댓글을 작성하여 작업을 선점할 수 있습니다.
  - 예: `제가 하겠습니다`

- **24시간 이내에 자원자가 없는 경우**,  
  이슈를 등록한 본인이 댓글을 통해 해당 작업을 맡을 수 있습니다.

---

## 2. 작업 기한 규칙
작업 선점 이후, 아래 기한 내에 작업을 완료해야 합니다.

-  **문서 작업**
  - 선점 댓글 작성 후 **24시간 이내 완료**

-  **기능 개발 / 버그 수정**
  - 선점 댓글 작성 후 **48시간 이내 완료**

---

## 3. 기타 사항
- 기한 내 작업이 진행되지 않을 경우,  
  다른 사람이 해당 이슈를 다시 선점할 수 있습니다.
- 작업 시작 시 빠른 진행을 위해 관련 내용을 충분히 확인한 후 선점합니다.
